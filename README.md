## ğŸ¨ Customization

### Change Colors
Edit CSS variables in `style.css`:
```
:root {
  --accent: #3b82f6;        /* Primary color */
  --accent-hover: #2563eb;  /* Hover state */
  --success: #10b981;       /* Success color */
}
```

### Update Content
1. **Personal Info:** Edit `index.html` lines 50-100
2. **Projects:** Update project cards in lines 400-600
3. **Social Links:** Change URLs in footer section
